# 📑 Resume-Xray | The Intelligence Tier

> **Deep Semantic Insights for the Job Market.** 
> Stop being invisible to "Black Box" ATS systems. 

Resume-Xray is AI-driven career-tech platform designed to provide job seekers with the same semantic analysis used by recruiters. It bypasses traditional keyword-matching to reveal the "DNA" of your professional identity.

---

## 🚀 Deployment
Live at: [https://resumexray-gamma.vercel.app/](https://resumexray-gamma.vercel.app/)

---

## ✨ Features

- **🧠 Agentic Resume Parsing:** Utilizing state-of-the-art AI agents to extract deep semantic meaning, not just keywords.
- **💎 Luxury Light UI:** A high-performance, minimalist interface built with Tailwind CSS v4 and Framer Motion for a "premium" feel.
- **🛡️ Secure Neural Lock:** Integrated biometric-style visual proctoring and secure Firebase authentication.
- **📊 Semantic Dashboard:** Real-time data visualization of your career trajectory using Recharts.

## 🛠️ The Stack

| Layer          | Technology                                                                 |
| :------------- | :------------------------------------------------------------------------- |
| **Framework**  | [Next.js 15+](https://nextjs.org/) (App Router & Turbopack)                |
| **Logic**      | [React 19](https://react.dev/) & [TypeScript](https://www.typescriptlang.org/) |
| **Styling**    | [Tailwind CSS v4](https://tailwindcss.com/) & [Clsx](https://www.npmjs.com/package/clsx) |
| **Motion**     | [Framer Motion 12](https://www.framer.com/motion/)                         |
| **Backend**    | [Firebase 12](https://firebase.google.com/) (Auth & Storage)               |
| **Components** | [Radix UI](https://www.radix-ui.com/) & [Lucide](https://lucide.dev/)      |
